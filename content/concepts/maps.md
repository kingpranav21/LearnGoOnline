# Maps

## What you learned
- A map is a hash table: `map[keyType]valueType`.
- You can create a map with a literal: `map[string]int{...}`.
- Lookup: `ages["Ada"]`.

## Example

```go
ages := map[string]int{"Ada": 36, "Linus": 54}
fmt.Println(ages["Ada"])
```
