# `if` / `else`

## What you learned
- Conditions do **not** need parentheses.
- Blocks always use braces `{ ... }`.

## Example

```go
if x%2 == 0 {
  fmt.Println("even")
} else {
  fmt.Println("odd")
}
```

## Try this
- Change `x` and predict the output before running.
- Add an `else if` branch for `x == 0`.
